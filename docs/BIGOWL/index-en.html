BIGOWL Ontology: Description back to ToC

BIGOWL (BIG data analytics OWL2 ontology) acts as a formal schema for representing and consolidating knowledge in Big Data analytics. Knowledge incorporation is, in turn, beneficial for efficient algorithmic performance by taking part in the operatorâ€™s design, parameter selection, and human interactive and decision-making strategies.
